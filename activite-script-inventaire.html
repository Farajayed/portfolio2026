📋 Script d'inventaire automatique PowerShell

Catégorie : Automatisation & Scripts

← Retour aux activités

Technologies

PowerShell GLPI WMI

Résultat clé

10h/semaine économisées

Périmètre

550 postes – 3 sites IUT

📋 Contexte

La mise à jour de l'inventaire du parc informatique dans GLPI était effectuée manuellement par les techniciens, poste par poste. Cette tâche chronophage (environ 10 heures par semaine) mobilisait des ressources qui auraient pu être allouées au support utilisateur.

L'objectif était de développer un script PowerShell capable de collecter automatiquement les informations matérielles et logicielles de tous les postes du réseau, puis de les exporter vers GLPI sans intervention manuelle.

⚙️ Développement du script

  • Analyse des besoins – Identification des données à collecter (modèle, RAM, disque, OS, logiciels installés, utilisateur connecté)
  • Requêtes WMI/CIM – Utilisation des classes Win32 pour interroger les machines à distance
  • Parcours du réseau – Script itérant sur la liste des hôtes actifs du domaine
  • Export GLPI – Formatage des données et mise à jour via l'API GLPI ou import CSV
  • Gestion des erreurs – Logs des machines injoignables ou en erreur
  • Planification – Tâche planifiée Windows pour exécution automatique hebdomadaire

💻 Extrait illustratif du script

# Collecte des informations matérielles via WMI
$computers = Get-ADComputer -Filter * | Select -ExpandProperty Name

foreach ($computer in $computers) {
    if (Test-Connection $computer -Count 1 -Quiet) {
        $sys   = Get-CimInstance -ComputerName $computer Win32_ComputerSystem
        $os    = Get-CimInstance -ComputerName $computer Win32_OperatingSystem
        $disk  = Get-CimInstance -ComputerName $computer Win32_LogicalDisk -Filter "DeviceID='C:'"

        # Export vers CSV pour import GLPI
        [PSCustomObject]@{
            Nom     = $computer
            RAM_Go  = [math]::Round($sys.TotalPhysicalMemory / 1GB, 1)
            OS      = $os.Caption
            Disque  = [math]::Round($disk.Size / 1GB)
        }
    }
} | Export-Csv "inventaire.csv" -NoTypeInformation -Encoding UTF8
